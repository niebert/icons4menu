Scan SVG Files for Navigation Menu

Last Update: 2021/03/12

This shell script wget_icons.sh of the repository Icons4Menu collects all SVG icons files and generates an index.html for all scanned files. The Icons can be downloaded with a wget shell script for Linux and MacOSX. The script can be downloaded and modified to select just a specific subset of the icons. The licenses for the different listed icons can view on the README.md of the repository icons4menue.
